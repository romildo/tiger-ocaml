# tiger-compiler-ocaml

A compiler for the Tiger programming language targetting LLVM and implemented in OCaml


## How to clean

```
$ dune clean
```

## How to compile

```
$ dune build src/driver.exe
```

## How to test the compiler

```
$ dune exec src/driver.exe
```
